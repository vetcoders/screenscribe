## JetBrains Mono

The Pro HTML report embeds a subset of **JetBrains Mono** (Latin + Latin-Extended,
weights 400 and 500) inline as a base64 `@font-face` in
`screenscribe/html_pro_assets/styles/screenscribe-theme.css`, so the report's
signature monospace renders identically offline without a font install or network
fetch.
