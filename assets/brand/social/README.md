## Setting the GitHub social preview (manual)

`og-b-flat.png` is the Open Graph card shown when the repo link is shared
(Slack, X, LinkedIn, Discord, etc.). GitHub can't pick it up from the repo —
upload it by hand:

**Repo → Settings → Social preview → Edit → upload `og-b-flat.png`.**

The same 1200×630 file works as the `og:image` for any future landing page.
